pub fn sample_cdf<'a>(
    cdf: &'a Vec<StoredExcite>, 
    imp_sample_dist: &ImpSampleDist, 
    max_cdf: Option<f64>, 
    rand: &mut Rand
) -> Option<(&'a StoredExcite, f64)>
Expand description

Sample a CDF (in decreasing order) by sampling a uniform random number up to max_cdf and binary searching the CDF cdf is a vector of StoredExcites, and we access their stored cumulative sums depending on imp_sample_dist max_cdf is chosen such that CDF(elem) = max_cdf.unwrap() for the first elem that is a valid sample (max_cdf = None samples the whole distribution) Returns the sampled excite and the probability of the sample O(log M)
