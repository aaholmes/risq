§Display Formatting Utilities (utils::display)

This module provides implementations of the std::fmt::Display trait for various custom data structures used in the risq crate, allowing them to be easily printed in a human-readable format (e.g., using println!("{}", my_struct)).

It also includes helper functions for formatting specific types (like bitstrings) and methods attached to VarWf for printing wavefunction information.
